\begin{letter}{Editor-in-Chief \\ Journal of Information Security and Applications}

\opening{Dear Editor,}

We submit our manuscript entitled ``$0.60 Is All You Need: Cost-Efficient LLM Fine-Tuning for SOC'' for consideration in \textit{Journal of Information Security and Applications}.

\textbf{Summary.} This paper presents cost analysis, $0.60 training cost, Pareto frontier. Our study is part of a systematic investigation into LLM fine-tuning for Security Operations Center (SOC) alert classification, where label compliance---the ability to produce only schema-valid labels---is critical for automated threat detection.

\textbf{Key contributions:}
\begin{itemize}
\item Rigorous experimental methodology: 5-seed validation, clean zero-overlap data splits, and dual-metric evaluation (strict vs.\ normalized F1).
\item Practical implications for SOC deployment and reproducible research.
\item Open-source code and datasets at \url{https://github.com/nutthakorn7/soc-llm-research}.
\end{itemize}

\textbf{Scope alignment.} This work addresses JISA's focus on information security through empirical evaluation of LLM fine-tuning for domain-specific classification.

This manuscript has not been published elsewhere and is not under consideration by another journal. All authors have approved the manuscript.
